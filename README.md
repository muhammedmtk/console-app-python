# Client Data Entry Console App

This is a simple Python console application developed as part of my learning journey in Python.

## Project Idea
The application allows the user to enter and manage client data through the terminal in a simple and interactive way.

## Features
- Add new client data
- Store basic client information
- Display entered data through the console
- Practice working with Python basics such as:
  - Variables
  - Conditions
  - Loops
  - Functions
  - Lists / Dictionaries

## Technologies Used
- Python

## Purpose
This project was built for practice and learning purposes to improve my understanding of Python fundamentals and console-based applications.

## How to Run
1. Make sure Python is installed on your machine.
2. Download or clone the project.
3. Open the project folder in the terminal.
4. Run the following command:

```bash
python main.py
